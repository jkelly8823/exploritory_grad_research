VULNERABILITY_DESCRIPTION: SQL Injection
EXPLANATION:
The code directly inserts the value of `$_COOKIE["mid"]` into the SQL query without proper sanitization. This allows an attacker to manipulate the query and potentially execute arbitrary SQL code. For example, an attacker could set the `mid` cookie to a value like `1'; DROP TABLE messages; --` which would drop the `messages` table.
